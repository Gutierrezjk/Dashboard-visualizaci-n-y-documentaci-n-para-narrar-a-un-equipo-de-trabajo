# Dashboard de salarios – Visualización y documentación

Este repositorio contiene la documentación y visualizaciones asociadas
al dashboard de análisis salarial desarrollado en Looker Studio, correspondientes a la actividad "Dashboard: visualización y documentación para narrar a un equipo de trabajo"

El objetivo del proyecto es analizar ingresos salariales normalizados
por género, industria y país, a partir de una base de datos de encuestas.

##  Dashboard
El dashboard interactivo puede consultarse en el siguiente enlace:
https://lookerstudio.google.com/s/qo6cozZ1xco
